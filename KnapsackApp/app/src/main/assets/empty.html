Welcome to Knapsack!

Knapsack is an offline web browser that saves web pages so that you can read them when you don't have internet connectivity like when you are on the subway or airplane.

When you save web pages to your knapsack, they will show up here!

Simple interface for the best reading experience

Save web pages to Knapsack from the "Open With" dialog anywhere in Android!

Share web pages from any browser to save in Knapsack!

PRO:Install the Chrome Plugin to remotely sync from desktop Chrome browser!

PRO:Setup Cloud Sync to backup your knapsack to the cloud and sync across devices!

PRO:Ad blocking powered by Adblock's Easylist!

Pause saving for Wi-Fi to minimize mobile data use!
